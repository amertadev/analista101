# About

analista101 dibangun oleh Direktorat Jenderal Analisisis Data - [Kementerian Riset dan Data - Badan Eksekutif Mahasiswa Universitas Jenderal Soedirman](https://dataatunsoed.bem-unsoed.com/) pada bulan Maret 2024. analista101 bukanlah program kerja resmi dari RISDAT, melainkan sebuah project inisiatif dari Analista 4.0 yang ingin mempermudah teman-teman dalam mengerjakan program kerja analisis data.

## Team

Untuk saat ini, project analista101 dikerjakan oleh [Jehian](https://sendomoka.dev) sebagai <span style={{ display: "inline-flex", alignItems: "center" }}><GiChessKing />Kadit</span> Analista 4.0. Kamu bisa melihat kontribusi dari kontributor lainnya di Github Community [amerta.io](https://github.com/amerta-io). analista101 bisa rilis berkat dukungan dari teman-teman risdat dan kakak-kakak risdat 1 sampai 3. Terima kasih banyak!
