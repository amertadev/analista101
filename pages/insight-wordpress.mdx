# Overview

Insight Wordpress adalah analisis pengunjung website [dataatunsoed.bem-unsoed.com](https://dataatunsoed.bem-unsoed.com) yang menggunakan plugin MonsterInsights, data ini up to date akan otomatis melakukan scraping setiap akhir bulan.
